# Changelog

All notable changes to the Tarvo Lilja Portfolio Website will be documented in this file.

## [1.0.0] - 2025-11-24

### Added

- Initial portfolio website implementation
- Fixed navigation bar with smooth scrolling
- Hero section with profile image and call-to-action buttons
- Education section with degree details and thesis information
- Skills section with categorized technical and soft skills
- Projects section with 3 project cards
- Contact section with professional links
- Fully responsive design for all screen sizes
- Modular CSS architecture with separate files for each section
- Modular JavaScript with ES6 modules
- Smooth scroll animations and transitions
- Active section highlighting in navigation
- Mobile-friendly hamburger menu
- Professional hover effects throughout
- Footer section

### Technologies

- HTML5 with semantic markup
- CSS3 with CSS Variables, Flexbox, and Grid
- Vanilla JavaScript ES6+ with modules
- Mobile-first responsive design approach

### File Structure

- 6 modular CSS files (navbar, hero, education, skills, projects, contact)
- 5 JavaScript modules (navbar, smoothScroll, activeSection, animations, contact)
- Clean separation of concerns for maintainability
